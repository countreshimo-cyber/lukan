## 🚀 Быстрый старт

### 1. Переименуйте главный файл
```
dr-lukan-website.html → index.html
```

### 2. Создайте структуру:
```
ваш-репозиторий/
├── index.html
├── endoskopicheskaya-hirurgiya.html
├── septoplastika.html
├── lechenie-hrapa.html
├── tonzillektomiya.html
├── lechenie-otitov.html
├── lechenie-rinitov.html
├── adenoidy.html
├── publikatsii.html
└── images/
    ├── doctor-portrait.jpg
    ├── clinic-interior.jpg
    ├── cert-diploma-kmn.jpg
    ├── cert-specialist.jpg
    ├── cert-diploma-base.jpg
    └── cert-conference.jpg
```

### 3. Загрузите на GitHub
```bash
git add .
git commit -m "Initial website upload"
git push origin main
```

### 4. Активируйте GitHub Pages
1. Зайдите в Settings вашего репозитория
2. Pages → Source: main branch, / (root)
3. Save

Сайт будет доступен через 2-3 минуты по адресу:
```
https://ваш-username.github.io/название-репозитория/
```
